# pe-interpolator
Generate instances with two .sfd files.

Run the python script to generate your fonts.

```
$ python3 interpolate.py yourFontLight.sfd yourFontHeavy.sfd YourSpecFile
```

Default format of creating a spec file, Ex: see default-spec file
```
FontFamilyName, FontName, FontFullName, WeightName, Weight
```
